# @chatool/\* component catalog

Interactive documentation for the `@chatool/*` packages, rendered straight from
their published `dist` — exactly what a consuming app gets from npm.

- **UI** — `@chatool/ui` components (`Button`).
- **Icons** — the `@chatool/icons` set.
- **Tokens** — the `@chatool/core` design tokens, light and dark.
- **Core** — `@chatool/core`'s `ChatoolProvider` + `useTheme`.

## Light / dark

Use the **theme toggle** in the toolbar (the paint-roller icon) to flip every
story between light and dark. It sets the `.dark` class that `@chatool/core`
keys its tokens off — the same mechanism `@chatool/core` drives in a real app.

## Running locally

```bash
pnpm build       # build the @chatool/* dist the stories import
pnpm storybook   # → http://localhost:6006
```

For live component editing, run `pnpm dev` (tsdown watch) in another terminal —
Vite HMR picks up the rebuilt `dist`.
